# ElevateFlow AI – Projektstatus & Todo

## ✅ Klart

### Kärnfunktionalitet
- Svensk AI-agent med dynamisk prompt (tjänsteanpassad)
- SMS + Email tvåvägskonversation
- Meta Ads webhook
- Automatisk sammanfattning till hantverkaren
- Retry-mekanism via Inngest

### Administrationsverktyg
- Företagshantering med tjänsteval (14 tjänster)
- API-nycklar per företag
- Dashboard med leads och metrics

### Frontend
- Modern Layout + shadcn/ui komponenter (Button, Card, Dialog, Table, etc.)
- Toast-meddelanden (Sonner)
- Ikoner (Lucide)

### Säkerhet & Skalbarhet
- Row Level Security (RLS)
- API-nyckel autentisering
